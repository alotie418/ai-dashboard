Frequently asked questions

What is SoloLedger?

A desktop app for recording sales and purchases, tracking products, inventory, receivables and payables, generating business documents, and viewing management-basis financial summaries. It is a bookkeeping and management-analysis tool — not an official tax-filing, audit, or statutory-compliance system.

Where is my data stored? Is it private?

All data is stored locally in a SQLite database inside the app's private Application Support directory on your Mac. SoloLedger does not require an account and does not upload your bookkeeping data to any server. See the Privacy Notice.

Does the Mac App Store version need an API key or internet connection?

No. The Mac App Store version runs fully offline for core bookkeeping. It does not ask for, store, or use any external API key.

How do I back up or move my data?

Use Settings → Data Backup to export a backup (database plus attachments) to a folder you choose, and to restore from a previous backup. Keep your own copies of exported backups; they are your data.

How do I change the app language?

Open Settings → Language and choose from the supported languages. The interface language is independent of the accounting profile.

How do I reopen the window after closing it?

Choose Window → SoloLedger from the menu bar (⌘0), or click the app icon in the Dock. Your saved data is never affected by closing or reopening the window.

数据存在哪里？是否私密？

全部数据以本地 SQLite 数据库形式保存在 Mac 上应用专属的 Application Support 目录中。 应用无需登录，也不会把你的账本数据上传到任何服务器。详见 隐私说明。

Mac App Store 版本需要 API Key 或联网吗？

不需要。App Store 版本的核心记账功能完全离线运行，不索取、不保存、不使用任何外部 API Key。

如何备份或迁移数据？

在 设置 → 数据备份 中导出备份（数据库 + 附件）到你选择的文件夹，并可从既有备份恢复。 请自行妥善保存导出的备份文件。

关闭窗口后如何重新打开？

在菜单栏选择 窗口 → SoloLedger（⌘0），或点击 Dock 中的应用图标。 关闭与重开窗口都不会影响已保存的数据。
