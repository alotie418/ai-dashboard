### 2. #226–#229 处理内容
- **#226**：手动 Model ID 输入成为设置页**主入口**；已配置 Key 的 provider 可**不重输 Key 直接测试连接**（后端用本地已存 Key，明文不出主进程）。
- **#227**：provider 测试失败**错误分类与呈现**——auth / permission / quota / modelNotFound / badRequest / serverError / network + **emptyResponse**（HTTP 2xx 成功但模型无可用内容）；展示**脱敏 providerMessage** 摘要，不再只显示泛化「AI 请求失败」。
- **#228**：默认模型改为真实验收可连——**DeepSeek `deepseek-v4-pro` → `deepseek-chat`**、**Kimi `kimi-k2.6` → `moonshot-v1-128k`**（不删任何模型；GLM/Doubao/Qwen/Gemini 默认不变）。
